# Syncnote-alpha

Syncnote-alpha is an AI-powered tool that transforms meeting notes into:

- 📄 Clear summaries  
- ✅ Actionable to-do lists  
- ✉️ Ready-to-send emails  

## 🚀 Features

- Paste meeting notes and analyze instantly
- Automatically extract key points
- Generate structured tasks
- Create professional email drafts

## 🛠 Tech Stack

- Node.js
- OpenAI API
- Express
